#
Callback after the File Group Upload Complete

POST /media/api/v1/workspaces/{workspace_id}/group-upload-callback

Parameters
Name In Type Required Description
workspace_id path string true Workspace ID
x-auth-token header string true Access Token
body body storage.FolderUploadCallbackInput true Body parameter
Responses
Status Meaning Description Schema
200 OK
open in new window
 OK storage.FolderUploadCallbackOutput

Example responses

{
 "code":0,
 "data":{},
 "message": "success"
}
