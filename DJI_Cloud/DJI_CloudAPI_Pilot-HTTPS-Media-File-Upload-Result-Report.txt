#
Report Media Upload Result

POST /media/api/v1/workspaces/{workspace_id}/upload-callback

Parameters
Name In Type Required Description
workspace_id path string true workspace id
x-auth-token header string true access token
body body media.UploadCallbackInput true
Responses
Status Meaning Description Schema
200 OK
open in new window
 OK media.UploadCallbackOutput

Example responses

{
 "code":0,
 "message":"success",
 "data":{
 "object_key":"5asjwu24-2a18-4b4b-86f9-3a678da0bf4d/example.jpg"
 }
}
