RTK Calibration Result

Topic: thing/product/{gateway_sn}/events

Direction: up

Method: rtk_calibration

Data:

Column Name Type constraint Description
output Output struct Calibration execution output information
» ext Additional Information struct Additional information
»» devices Device List array Calibration equipment results list
»»» sn Device Serial Number string Device Serial Number（SN）
»»» type Calibration Type int {1: Manual Calibration} Calibration type
»»» module Module string { "3": Dock, "6": Relay } The module to which the device belongs
»»» result result Code int Calibration result code（0 ：Success，not 0 ：Failed）
»»» status Device Status string {"canceled":"cancellation or termination","failed":"Failed","in_progress":"executing","ok":"","paused":"pause","rejected":"reject","sent":"already sent","timeout":"time out"} Equipment calibration status（Timeout return "failed" and provide error code）
»» version Calibration Version int Calibration version
» progress Schedule struct Calibrate the overall progress, the device will only push results when they are available
»» percent Progress Percentage int Progress percentage, contains only two values，o is 100，failed is 0）
»» current_step Current Step int fixed value 1
» status Task Status enum_string {failed":"Failed","ok":"Success"} Returns ok when all devices are calibrated successfully,If any device fails, it returns failed.
result Return Code int fixed value is 0，Check the error code of the specific calibration device in the calibration result list

If the value of the drone_in_dock field that is pushed by the device property is 0, and the aircraft is confirmed to be out of the dock through the dock camera, this command can be called to force the dock cover to close.Otherwise, the propellers of aircraft may be pinched.

Topic: thing/product/{gateway_sn}/services

Direction: down

Method: cover_force_close

Data: null

Topic: thing/product/{gateway_sn}/services_reply

Direction: up

Method: cover_force_close

Data:

Column Name Type constraint Description
result Return code int Non-zero represents an error
output Output struct
»status Task state enum_string {"canceled":"Cancelled or terminated","failed":"Failed","in_progress":"Executing","ok":"Executed successfully","paused":"Paused","rejected":"Rejected","sent":"Issued","timeout":"Timeout"}
