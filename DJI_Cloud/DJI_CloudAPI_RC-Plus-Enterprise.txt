# Device Property List

pushMode:

0: Stable frequency data. Device will report in the frequency of 0.5HZ. (Topic:thing/product/{device_sn}/osd)

1: State data. It is reported when the state changes. (Topic:thing/product/{device_sn}/state)

accessMode:

r: Property can only be read

rw: Property can be read and written (Topic:thing/product/{gateway_sn}/property/set)
