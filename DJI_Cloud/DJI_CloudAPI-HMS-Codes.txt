# Error Code Explanation
Overall, the error code follows the format of "ABCDEF". A represents the source. BC represents the function module. DEF represents the module customized error.
The error code description of HMS can not be directly obtained through the error code. Developer needs to complete the concatenation of the "Copy Key" according to the rules introduced in the HMS Function open in new window . Search the error code description in the hms.json open in new window based on the "Copy Key". If there are variables in the description, developer needs to replace them according to the rules.
If the following error code description can not solve your problem, we provide Error Code Supplementary Reading open in new window .
